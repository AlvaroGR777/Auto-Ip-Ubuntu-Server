
El script en Python solicita al usuario ingresar una dirección IP, una máscara de red y una dirección de gateway. Luego, valida estos valores asegurándose de que estén en el formato correcto y dentro del rango válido. Una vez validados, aplica la nueva configuración de red en una máquina Ubuntu Server utilizando comandos de línea de comandos, desactivando y luego activando la interfaz de red con los nuevos parámetros.

-------

The Python script prompts the user to enter an IP address, a subnet mask, and a gateway address. It then validates these values to ensure they are in the correct format and within the valid range. Once validated, it applies the new network configuration on an Ubuntu Server machine using command-line commands, by deactivating and then activating the network interface with the new parameters.
